# Experts methods

## React + TypeScript + Vite 

## Requirements

- [NodeJS](https://nodejs.org)

## Getting start
- Install PNPM
```shell
npm i -g pnpm@latest
```
- Install dependencies
```shell
pnpm i
```
- You can build the project
```shell
pnpm build
```
and watch ``index.html`` in ``dist`` folder
- OR start in development mode
```shell
pnpm dev
```
and watch on http://localhost:5173/
